# Theme Presets

Use these presets when the user asks for three distinct, professional themes with tasteful graphics.

## 1) Krakow Guild Hall

### Theme concept
- **Vibe:** Historic Polish craft guild hall with warm materials and modern clarity.
- **Adjectives:** heritage, confident, crafted, grounded, refined
- **Never do:** candy palettes, glowing neon, playful sticker motifs

### Token starter set
- **Colors:**
  - `bg`: `#F6F2EA`
  - `surface`: `#FFFCF7`
  - `text`: `#2B241A`
  - `muted`: `#6A5E4C`
  - `primary`: `#7A1E2B`
  - `secondary`: `#274C77`
  - `accent`: `#B88746`
  - `success`: `#2E6A4A`
  - `warning`: `#B2742A`
  - `danger`: `#A23232`
- **Typography:** Display = dignified serif, Heading = humanist sans, Body = readable sans, Mono = practical mono.
- **Shapes:** Radius 8-14px, restrained borders, soft material shadows.

### Graphics direction
- Heraldic line motifs, subtle seal marks, engraved divider lines.
- Background motif: linen/parchment texture with low-opacity emblem pattern.

### Motion plan
- **Profile:** calm.
- **Durations:** fast 120ms, medium 200ms, slow 300ms.
- **Easing:** `cubic-bezier(0.2, 0.7, 0.2, 1)`.
- **Micro-interactions:** gentle button lift, border tint transitions, subtle panel fade.
- **Reduced motion:** no lift; color-only state transitions.

### Component feel
- **Button:** deep burgundy primary with brass accent outline on focus.
- **Card:** parchment-like panels with disciplined spacing.
- **Header:** refined strip with emblem-friendly left area.

## 2) Arcane Collegium

### Theme concept
- **Vibe:** scholarly magical academy; elegant, mysterious, and structured.
- **Adjectives:** scholarly, arcane, precise, atmospheric, premium
- **Never do:** cartoon stars, oversaturated gradients, playful bounce animations

### Token starter set
- **Colors:**
  - `bg`: `#12111B`
  - `surface`: `#1C1A2A`
  - `text`: `#ECE9F8`
  - `muted`: `#AAA2C7`
  - `primary`: `#6A5ACD`
  - `secondary`: `#2B6F8E`
  - `accent`: `#C6A15B`
  - `success`: `#3B8D6C`
  - `warning`: `#C8963C`
  - `danger`: `#A64B5D`
- **Typography:** Display = serif title style, Heading = modern sans, Body = neutral sans, Mono = compact mono.
- **Shapes:** Radius 8-12px, layered depth, fine luminous borders.

### Graphics direction
- Constellation diagrams, manuscript ornaments, geometric linework.
- Background motif: subtle star-map/geometry texture in low opacity.

### Motion plan
- **Profile:** calm-lively hybrid.
- **Durations:** fast 130ms, medium 230ms, slow 360ms.
- **Easing:** `cubic-bezier(0.16, 1, 0.3, 1)`.
- **Micro-interactions:** soft shimmer on key CTA, section fade transitions, quiet progress sweep.
- **Reduced motion:** remove shimmer movement; keep static highlight changes.

### Component feel
- **Button:** deep violet with restrained metallic edge.
- **Card:** dark velvet panels with thin elegant borders.
- **Header:** academy banner style with calm glow accents.

## 3) Metropolitan Workshop

### Theme concept
- **Vibe:** modern city workshop; practical, energetic, and contemporary.
- **Adjectives:** modern, industrious, clean, strong, technical
- **Never do:** neon nightclub styling, excessive glow, toy-like iconography

### Token starter set
- **Colors:**
  - `bg`: `#14181F`
  - `surface`: `#1D2430`
  - `text`: `#EAF0F7`
  - `muted`: `#9AA7B8`
  - `primary`: `#1F6AA5`
  - `secondary`: `#3F7A5B`
  - `accent`: `#C8893C`
  - `success`: `#3E8B61`
  - `warning`: `#B98A3A`
  - `danger`: `#B14E4E`
- **Typography:** Display = strong sans, Heading = engineered sans, Body = neutral sans, Mono = code-friendly mono.
- **Shapes:** Radius 6-10px, crisp lines, structured shadows.

### Graphics direction
- Blueprint lines, architectural overlays, workshop stamp marks.
- Background motif: subtle blueprint grid/diagram texture.

### Motion plan
- **Profile:** precise.
- **Durations:** fast 110ms, medium 180ms, slow 260ms.
- **Easing:** `cubic-bezier(0.24, 0.9, 0.32, 1)`.
- **Micro-interactions:** crisp focus rings, subtle slide-in indicators, measured hover elevation.
- **Reduced motion:** remove movement, keep instant state and contrast changes.

### Component feel
- **Button:** matte steel-blue primary with clear active states.
- **Card:** structured utility cards with functional hierarchy.
- **Header:** compact command-bar style top section.

## Uniqueness matrix (quick check)

Ensure all three differ across all major axes:

- **Palette logic:** Neon dark, pastel light, cosmic deep.
- **Typography personality:** heritage serif-sans, arcane serif-sans, engineered sans.
- **Shape language:** warm craft panels, elegant academy layers, crisp urban structure.
- **Graphics style:** heraldic craft, manuscript geometry, blueprint workshop.
- **Motion personality:** calm, atmospheric, precise.
- **Texture/background:** linen emblem, star-map manuscript, technical blueprint.
